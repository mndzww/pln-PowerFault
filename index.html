// TENTANG SISTEM
Klasifikasi Otomatis Gangguan Transmisi

Sistem ini dikembangkan untuk mengotomatisasi proses analisis gangguan pada jaringan transmisi tenaga listrik 150 kV. Dengan memanfaatkan data rekaman Digital Fault Recorder (DFR) dalam format standar COMTRADE, model CNN 1-D mengekstraksi pola temporal dari sinyal arus dan tegangan untuk mengidentifikasi jenis dan lokasi gangguan secara cepat dan akurat.

⚡
COMTRADE INPUT

Mendukung format standar IEEE C37.111 COMTRADE dengan parsing otomatis file .cfg, .dat, dan .hdr.

🧠
CNN 1-D ENGINE

Arsitektur Convolutional Neural Network 1 Dimensi yang dioptimasi untuk pemrosesan time-series sinyal listrik.

🔍
8 KELAS GANGGUAN

Flashover, 1Φ-G, 2Φ, 2Φ-G, 3 Fasa, Mekanis, Overload, dan Normal.

📋
REKOMENDASI TINDAKAN

Output analisis dilengkapi rekomendasi penanganan berdasarkan jenis gangguan yang terdeteksi.

📊
VISUALISASI SINYAL

Menampilkan waveform sinyal arus dan tegangan dari semua fasa dengan anotasi titik gangguan.

📄
LAPORAN PDF OTOMATIS

Generate laporan analisis PDF multi-template siap dokumentasi teknis dan pelaporan ke PLN pusat.
